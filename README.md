# AGROMARGEN 2026-27

Calculadora de márgenes agrícolas para la campaña 2026-27 en Argentina.

## Características

- **Cálculo de márgenes** para 7 cultivos/secuencias:
  - Soja de primera
  - Trigo + Soja de segunda
  - Trigo
  - Maíz Temprano
  - Maíz Tardío
  - Sorgo
  - Girasol

- **Panel de decisiones**: Análisis completo por cultivo
- **Break-even**: Punto de equilibrio financiero
- **Bonificaciones de calidad**: Ajustes por calidad del grano
- **Cobertura de precios**: Estrategias de venta forward
- **Análisis de sensibilidad**: Variación de precios y rindes
- **Gráficos interactivos**: Visualización con Chart.js
- **Exportación**: PDF y Excel
- **Persistencia**: Guardar y cargar escenarios
- **Actualización de precios**: Fetch automático desde BCRP
- **Modo presentación**

## Uso

Simplemente abre `AGROMARGEN_2026-27.html` en cualquier navegador moderno. No requiere instalación ni servidor.

## Tecnologías

- HTML/CSS/JS vanilla
- Chart.js + chartjs-plugin-annotation
- jsPDF + jsPDF-autotable
- SheetJS (Excel)
- Google Fonts (Manrope)
